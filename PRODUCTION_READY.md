## 📱 الصفحات والميزات

### 1. الصفحة الرئيسية (`/`)
- عرض مباريات اليوم مقسمة حسب الدوري
- قسم خاص للمباريات المباشرة في الأعلى (مع نقطة حمراء تنبض)
- بطاقات المباريات تحتوي على:
  - شعار الفريقين
  - النتيجة (للمباريات المنتهية أو الجارية)
  - وقت المباراة (للمباريات القادمة)
  - حالة المباراة (مباشر / انتهت / لم تبدأ)
- **تحديث تلقائي كل 60 ثانية**

### 2. صفحة الترتيب (`/standings`)
- عرض ترتيب جميع الدوريات الـ 9
- جداول مفصلة مع:
  - المركز
  - اسم الفريق
  - عدد المباريات
  - الفوز / التعادل / الخسارة
  - الأهداف (له - عليه)
  - فارق الأهداف
  - النقاط
- تلوين المراكز:
  - **أخضر**: مراكز دوري الأبطال (1-4)
  - **أحمر**: مراكز الهبوط (الـ 3 الأخيرة)

### 3. صفحة الهدافين (`/scorers`)
- عرض أفضل 10 هدافين في كل دوري
- معلومات مفصلة:
  - اسم اللاعب
  - الجنسية
  - الفريق (مع الشعار)
  - عدد الأهداف
  - عدد التمريرات الحاسمة (Assists)

### 4. صفحة البطولات (`/competitions`)
- عرض جميع الدوريات الـ 9
- الضغط على أي دوري يأخذك لصفحة تفصيلية

### 5. صفحة تفاصيل المباراة (`/matches/:id`)
- معلومات كاملة عن المباراة:
  - الفريقان والنتيجة
  - اسم البطولة
  - الملعب
  - الحكم
  - أحداث المباراة (الأهداف، البطاقات، التبديلات)
  - التشكيلة الأساسية
  - الإحصائيات الكاملة

## 🚀 كيفية النشر إلى Cloudflare Pages

### 1. إعداد Cloudflare
```bash
# تسجيل الدخول
npx wrangler login

# إنشاء مشروع Pages
npx wrangler pages project create koorax \
  --production-branch main
```

### 2. بناء المشروع
```bash
npm run build
```

### 3. النشر
```bash
npm run deploy
# أو
npx wrangler pages deploy dist --project-name koorax
```

### 4. إعداد المتغيرات البيئية
```bash
# إضافة توكن API
npx wrangler pages secret put SPORTMONKS_API_TOKEN \
  --project-name koorax

# القيمة: 4Q2MEVr3Ee1DnjWipeOAiq7mIlkbDlWRTx02XsPoeZUB0zMeh8EtkYLysMJP
```
