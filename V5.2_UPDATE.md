## 🔧 التفاصيل التقنية

### ملفات معدلة
1. **src/index.tsx**
   - إضافة تبويب "صناع الأهداف"
   - إضافة دالة `loadAssists()`
   - فلترة اللاعبين الذين لديهم assists > 0
   - ترتيب تنازلي حسب عدد الأسيست

2. **ARAB_LEAGUES_NOTE.md** (جديد)
   - توثيق شامل عن الدوريات العربية
   - شرح القيود الحالية
   - اقتراح حلول بديلة

### API المستخدم
- **Endpoint**: `/api/competitions/:id/scorers`
- **البيانات المستخدمة**: 
  - `scorer.assists` - عدد التمريرات الحاسمة
  - `scorer.player.name` - اسم اللاعب
  - `scorer.player.nationality` - الجنسية
  - `scorer.team.name` - اسم الفريق
  - `scorer.team.crest` - شعار الفريق

### معالجة الحالات الخاصة
```javascript
// فلترة اللاعبين الذين لديهم assists
const assistProviders = scorers
  .filter(scorer => scorer.assists && scorer.assists > 0)
  .sort((a, b) => b.assists - a.assists);

// إذا لم يوجد بيانات
if (assistProviders.length === 0) {
  container.innerHTML = 'لا توجد بيانات صناع الأهداف متاحة';
}
```
